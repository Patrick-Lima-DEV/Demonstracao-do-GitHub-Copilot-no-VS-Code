# Projeto: Demonstração do GitHub Copilot no VS Code

## 📌 Descrição do Projeto
Projeto didático para portfólio, mostrando como o GitHub Copilot pode ser utilizado para acelerar o desenvolvimento em Python no Visual Studio Code. Inclui exemplos práticos, guia de configuração e insights sobre produtividade com IA.

## 🚀 Tecnologias Utilizadas
- Python 3.x
- Visual Studio Code
- GitHub Copilot

## ⚙️ Como Configurar o Ambiente
1. Instale o [Python](https://www.python.org/downloads/)
2. Instale o [Visual Studio Code](https://code.visualstudio.com/)
3. Instale a extensão "GitHub Copilot" no VS Code:
   - Abra o VS Code
   - Vá em Extensões (Ctrl+Shift+X)
   - Busque por "GitHub Copilot" e clique em instalar
4. Faça login com sua conta GitHub

## 🤖 Como Usar o GitHub Copilot
- Ative o Copilot no VS Code (ícone no canto inferior direito)
- Comece a digitar comentários ou funções em Python
- Aceite sugestões com `Tab` ou `Enter`
- Experimente gerar funções, completar código e criar testes automaticamente
